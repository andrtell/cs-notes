Redirecting output and using shell commands.

----

__The program__

```c
// main.c

#include <stdio.h>

int main() {
    printf("Roses are red\n"
           "Violets are blue,\n"
           "Sugar is sweet\n"
           "And so are you.\n");
    return 0;
}
```

_compile_

```sh
cc -O0 -g -o main main.c
```

_debug_

```sh
gdb ./main
```

__GDB Session__


```
(gdb) run > poem.txt

[Inferior 1 (process 111246) exited normally]

(gdb) !cat poem.txt

Roses are red
Violets are blue,
Sugar is sweet
And so are you.
```
