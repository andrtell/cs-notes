__Program__

```c
// main.c

int main(void) {
	int product = 1;
	for (int n = 1; n <= 10; n++) {
		product *= n;
	}
	return 0;
}
```

_compile_

```sh
cc -O0 -g -o main main.c
```

_debug_

```sh
gdb ./main
```
